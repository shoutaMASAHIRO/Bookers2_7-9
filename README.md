# Bookers2 - X(Twitter) Style Book Sharing App

本アプリケーションは、ユーザー同士で本の情報を共有し、交流を深めるためのSNSプラットフォームです。
モダンなモバイルアプリの操作性と、X(Twitter)風の洗練されたデザインを兼ね備えています。

## 1. 動作環境
- **Ruby**: 3.2.10
- **Rails**: 6.1.7.10
- **Node.js**: v17以上対応 (OpenSSL Legacy Provider)
- **Database**: SQLite3
- **Main Gems**: Devise, Raty, Chart.js, Bootstrap, FontAwesome

---

## 2. 主要機能一覧

### ■ ユーザー・投稿機能
- **認証システム**: `devise`によるサインアップ、ログイン機能。
- **プロフィール管理**: 名前の変更、自己紹介、ActiveStorageによるプロフィール画像設定。
- **本の投稿**: タイトル、本文に加え、**5段階評価(星)**と**カテゴリタグ**を設定可能。
- **5段階評価 (Raty 4)**: 投稿時に直感的な星マークで評価。一度投稿した評価は変更不可。
- **カテゴリ検索**: 投稿に付与されたタグをクリックすることで、同じカテゴリの投稿を即座に検索。

### ■ インタラクティブな一覧・詳細
- **動的ソート機能**: 一覧画面にて「最新順」「評価の高い順」「トレンド（週間いいね順）」をリアルタイムに切り替え。
- **リアルタイムいいね**: 非同期通信(AJAX)により、ページをリロードせずにいいねの反映と並び替えが実行。
- **閲覧数カウント**: 各投稿の詳細ページへのアクセス数を自動集計し表示。
- **コメント機能**: 投稿に対する非同期でのリプライ（削除機能付き）。

### ■ コミュニケーション
- **フォロー・フォロワー**: ユーザー間のフォロー機能。
- **1対1 DM機能**: 相互関係に関わらず、**自身がフォローしているユーザー**とプライベートなチャットが可能（140文字制限）。
- **グループ機能**:
  - グループの作成・編集（作成者がオーナー）。
  - 自由な参加・脱退。
  - **イベント告知メール**: オーナーはグループメンバー全員に対して一斉にメール通知を送信可能。

### ■ データ分析・統計
- **投稿数統計**: 今日・前日・今週・先週の投稿数比較と前日比・前週比の自動計算。
- **7日間グラフ (Chart.js)**: 過去7日間の投稿推移を視覚的な折れ線グラフで表示。
- **特定日検索**: カレンダーから日付を選択し、その日の投稿数を非同期で取得。

## 3. UI/UX デザインの特徴
- **X(Twitter)風テーマ**: 白を基調とし、細いボーダーと円形アイコンを活用したタイムライン形式。
- **モバイル最適化**: 画面下部に固定の「ボトムナビゲーション」を配置し、スマホ操作に対応。
- **通知・エラーのダイアログ化**: バリデーションエラーや成功メッセージを、Bootstrapのモーダル（ポップアップ）形式で統一して表示。
- **マイクロインタラクション**: ボタンクリック時にわずかに縮小するアニメーションを付与し、操作感を向上。

## 4. 技術的な工夫・トラブルシューティング

### ① Windows環境への最適化
- **Spring/Pumaの安定化**: Windowsで不安定な`Spring`を停止・除去し、Pumaの自動再起動プラグインを無効化することでクラッシュを防止。
- **画像処理の回避**: Windows環境で導入が困難な`ImageMagick`に依存せず、CSS側でリサイズ処理を行うようにモデルを調整。

### ② モダンJavaScriptへの対応
- **Webpacker/OpenSSL**: Node.js v17以降のセキュリティ制限を回避するため、ビルド時に`openssl-legacy-provider`を強制適用。
- **Raty 4のVanilla JS化**: jQueryに依存しない最新のRaty 4を導入。RailsのWebpack環境で動作させるためにESモジュールのエクスポート・インポートを手動調整。

### ③ Turbolinksとの共存
- ページ遷移時にJavaScriptが発火しない問題を解決するため、すべての初期化処理（Chart.js, Raty, Modal等）を `turbolinks:load` イベントに紐づけ。
